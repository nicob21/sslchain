What is Sslchain?

Sslchain is a solution using blockchain technology to ensure a SSL certificate validity without trusting a certification authority (CA).

Currently, SSL certificates are signed by CA that need to be trusted by everyone. Sslchain propose a new way to manage the certificates taking advantage of the blockchain technology

Create a certificate

In order to store your certificate in the blockchain, please execute the following command:

# openssl req -new -newkey rsa:2048 -nodes -keyout <my_domain>.key -out <my_domain>.csr

Then self-sign the file .csr with your private key as follow.

# openssl x509 -req -days 365 -in <my_domain>.csr -signkey <my_domain>.key -out <my_domain>.crt
